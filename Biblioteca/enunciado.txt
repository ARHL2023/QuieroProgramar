En este proyecto, crearás un sistema de gestión de una biblioteca.
El objetivo es desarrollar una aplicación de consola en Java que permita realizar operaciones básicas, como agregar, buscar, eliminar y mostrar libros disponibles. Además, se incorporará herencia para introducir las clases Cliente y Administrador.

Requisitos del proyecto:

1.	Crea una clase Libro con los siguientes atributos: titulo, autor, codigo, cantidad y disponible.
    Asegúrate de implementar los métodos necesarios para acceder y modificar estos atributos.

2.	Crea una clase Persona como clase base, que contendrá los atributos comunes a Cliente y Administrador,
    como nombre y edad. Incluye los métodos de acceso y modificación necesarios.

3.	Crea una clase Cliente que herede de Persona. Agrega atributos adicionales, como numeroCliente y librosPrestados.
    Implementa los métodos necesarios para acceder y modificar estos atributos.

4.	Crea una clase Administrador que herede de Persona.
    Esta clase será responsable de ingresar nuevos libros a la biblioteca.
     Puedes agregar atributos adicionales, como codigoAdmin o rol, si lo consideras necesario.
     Implementa los métodos necesarios para acceder y modificar estos atributos.

5.	Crea una clase Biblioteca que contenga una colección de libros.
    Esta clase debe tener métodos para agregar libros, buscar libros por título o autor, eliminar libros y mostrar
    la lista de libros disponibles.

6.	Implementa un menú de consola para que los usuarios interactúen con la biblioteca.
    El menú debe tener las siguientes opciones:

    a. Iniciar sesión como cliente: el usuario ingresará su número de cliente para acceder
        a las funciones disponibles para los clientes, como consultar y tomar prestados los libros.

    b. Iniciar sesión como administrador: el usuario ingresará su código de administrador y contraseña para
        acceder a las funciones disponibles para los administradores, como ingresar nuevos libros a la biblioteca.

    c. Agregar un libro: esta opción solo estará disponible para el administrador.
        Se ingresarán los detalles del libro (título, autor, código, cantidad) y se agregará a la biblioteca.

    d. Buscar libro por título: el usuario ingresará el título y se mostrará una lista de libros con ese título.

    e. Buscar libro por autor: el usuario ingresará el autor y se mostrará una lista de libros escritos por ese autor.

    f. Mostrar lista de libros disponibles: se mostrará una lista de todos los libros disponibles en la biblioteca.

    g. Salir: el usuario podrá salir del programa.

7.	Utiliza las clases y métodos de las colecciones adecuadas (por ejemplo, ArrayList, HashMap) para almacenar y manipular los libros y los usuarios de la biblioteca.
